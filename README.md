## Django installation process
`
- .Ensure you have python in your system or download and install from   https://www.python.org/downloads/
- .  create backend folder and install virtual environment using this command        python -m venv
- . activate the virtual environment using       env\Scripts\activate.bat
`

## Install vite with react for your frontend
`
- .on your terminal use this command    npm create vite@latest frontend
- . cd frontend and run    npm install
- . npm install -D tailwindcss @tailwindcss/postcss postcss autoprefixer
- . npm run dev and open this localhost link on your browser   http://localhost:5173
`
